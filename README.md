# CEDRO TEST

## Clone the repo and install dependencies

### [GH Page](https://github.com/shibnev/CEDRO/dist/index.html)

```bash
git clone https://github.com/shibnev/CEDRO.git
cd CEDRO
npm i
or yarn
```

## Usage

### Development server

```bash
npm run dev
```

### Production build

```bash
npm run build
```
